
Nombre del proyecto: BLOCK 593

Integrantes: Rafael Chiriboga & Julian Amagua

Temática: Tienda de ropa y calzado de estilo urbano (Streetwear).

Descripción del proyecto: Es una página web creada para vender moda urbana. El sitio permite a los usuarios navegar por diferentes colecciones de ropa como zapatos, abrigos (hoodies) y accesorios. Nos enfocamos en que sea fácil de usar en celulares, que tenga un diseño moderno con colores verde y crema, y que incluya un formulario de contacto que funciona de verdad para recibir mensajes de los clientes.
